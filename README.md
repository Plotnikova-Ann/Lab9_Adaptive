## Описание проекта

В данной лабораторной работе изучаются медиа-запросы (Media Queries) — ключевая технология для создания адаптивных веб-страниц, которые корректно отображаются на всех устройствах: от мобильных телефонов до широкоформатных мониторов.
# Выполненные задания
## Базовый адаптивный макет (responsive.html)
Создан адаптивный макет, который включает:

- Header с логотипом и навигацией

- Hero секцию с приветствием

- Features секцию с тремя преимуществами

- Cards секцию с карточками товаров

- Footer с копирайтом

## Тестирование адаптивности
Проведено тестирование на трех разрешениях:

- Mobile (375px, iPhone SE) - вертикальное расположение

- Tablet (768px, iPad Mini) - двухколоночная сетка

- Desktop (1920px) - многоколоночная сетка
